Zeus Bot

  • Kaptian Core
  • Discord Bot

Zeus bot is a discord bot written in Javascript utilising multiple libraries, the main library being Discord.JS. Zeus has many features that are planned and already implemented, Zeus uses lavalink to provie users with a great quality music service.

Some of the many features planned for the bot are things such as per server configuration utilising Firebase, A Dashboard for both the user to setup the bot's settings per server and a developer dashboard with even more graphs than displayed on the user stats page.

The Zeus Support Server is where you can chat to the developers of the project as well as report bugs. Here is the Status Page for the bot if it is down for maintanence or issues keeping it down we will most likely update it here. We have a Development Trello which is public to display what we are currently working on and ideas we have in the backlog.
