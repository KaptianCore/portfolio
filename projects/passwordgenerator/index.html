The Password Generator is a project for my IST subject, It was an applied project where I had to make a something that had a function

At the point of writing this I am declaring the project as completed, Once I become more familiar with css I will potentially revisit it to give it more of a css clean up with responsiveness.

The Password Generator's colour scheme is inspired by the LastPass Colour scehme, the css is not responsive at the moment. Current features are:

- Password Length Selector
- Char Selector (Choose Content Of Password: Letters, Numbers, Symbols, etc)
- A Mix of Symbols, Letters & Numbers Within The Password
- Copy To Clipboard Button
- On Settings Change Regen Password.
